## Overview

This is a perspective skill for AI agents (Hermes, OpenClaw, Claude). When loaded, the agent adopts Dan Koe's mental models: Deep Generalist, Mental Entropy, Creator as Product, Attention Sovereignty.

---

## Structure

```
dan-koe-perspective/
├── SKILL.md           # Skill definition and activation
├── references/        # Source material, quotes, transcripts
└── examples/          # Usage patterns and prompts
```

---

## Installation

```bash
# Copy to agent's skills directory
cp -r dan-koe-perspective ~/.hermes/skills/
```

---

## Usage

Load the skill in any compatible agent:

```
load dan-koe-perspective
```

The agent will now respond using Dan Koe's first-person perspective and decision framework.

---

## Use Cases

- Prompt engineering — apply the Dan Koe lens to problem-solving
- Agent personality — give agents a distinct voice and worldview
- Creative reasoning — approach problems as a deep generalist
