## 📖 What Is This?

This is a **perspective skill** for AI agents (OpenClaw, Claude, etc.). When activated, the AI agent speaks as Dan Koe — adopting his first-person voice, mental models, and decision framework.

## 🎯 Use Cases

- **Prompt engineering inspiration** — adopt the Dan Koe lens
- **AI agent personality** — give your agent a unique voice
- **Creative problem solving** — approach problems like a deep generalist
